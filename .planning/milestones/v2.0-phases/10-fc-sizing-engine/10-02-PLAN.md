  <behavior>
    The implementation must satisfy these exact behaviors:

    Step 1 — Server totals:
    - totalServers = input.racks.reduce((sum, r) => sum + r.serverCount, 0)

    Step 2 — Per-fabric port demand:
    - hostPortsPerFabric = totalServers * Math.floor(input.hbaPortsPerServer / 2)
      (floor for odd HBA counts; with 2 HBAs this gives 1 port per fabric per server)
    - storagePortsPerFabric = Math.ceil((input.storageArrayCount * input.storageTargetPorts) / 2)

    Step 3 — Effective ports and switch count:
    - effectiveIslPerSwitch = Math.min(input.islPortsPerSwitch, SW.maxIslPorts)  [runtime clamp]
    - devicePortsPerSwitch = basePorts - effectiveIslPerSwitch  [ISL ports come from same pool]
    - For podLicenseUnit > 0 (fixed-port switches):
        extraPortsNeeded = portsNeededPerSwitch - basePorts
        podCount = Math.ceil(extraPortsNeeded / podLicenseUnit) [if extraPortsNeeded > 0, else 0]
        effectivePorts = Math.min(basePorts + podCount * podLicenseUnit, totalPorts)
    - For podLicenseUnit === 0 (directors): effectivePorts = totalPorts, podLicensesRequired = 0
    - fabricSwitchCount = Math.max(1, Math.ceil(hostPortsPerFabric + storagePortsPerFabric) / effectivePorts)
    - podLicensesRequired = podCountPerSwitch * fabricSwitchCount * 2  [both fabrics]

    Step 4 — ISL count (bandwidth fan-in formula, NOT switch count multiplier):
    - hostBandwidth = hostPortsPerFabric * SW.speedGbps
    - storageBandwidth = storagePortsPerFabric * SW.speedGbps
    - requiredIslBandwidth = Math.min(hostBandwidth, storageBandwidth) * (1 / FC_FAN_IN_MAX)
    - islCount = Math.max(2, Math.ceil(requiredIslBandwidth / SW.speedGbps))
    - islCables = islCount * 2  [both fabrics]

    Step 5 — Optics count:
    - totalLinksPerFabric = hostPortsPerFabric + storagePortsPerFabric + islCount
    - fcOpticsCount = totalLinksPerFabric * 2 * 2  [2 optics per cable × 2 fabrics]

    Step 6 — Ratios:
    - fanInRatio = storagePortsPerFabric > 0 ? hostPortsPerFabric / storagePortsPerFabric : 0
    - islOversubscriptionRatio = islCount > 0 ? hostPortsPerFabric / islCount : 0

    Step 7 — Violations:
    - FC_OVERSUBSCRIPTION_EXCEEDED: if fanInRatio > FC_FAN_IN_MAX → { code, ratio: fanInRatio, maxRatio: FC_FAN_IN_MAX }
    - FC_PORT_SATURATION: if (hostPortsPerFabric + storagePortsPerFabric) > (fabricSwitchCount * effectivePorts)
        → { code, requiredPorts: hostPortsPerFabric + storagePortsPerFabric, availablePorts: fabricSwitchCount * effectivePorts }
    - FC_ISL_UNDERPROVISIONED: if islCount < 2 [can only happen if Math.max(2, ...) is somehow bypassed — defensive]
        → { code, islsAvailable: islCount, islsRequired: 2 }

    Return (INVARIANT: fabricBSwitches MUST equal fabricASwitches):
    - fabricASwitches: fabricSwitchCount
    - fabricBSwitches: fabricSwitchCount  [identical — dual-fabric symmetry]
    - islPortsPerFabric: islCount
  </behavior>
