<objective>
Unify the Ethernet schema, stores, and mode selector to support both Clos and Three-Tier topologies under a single "Ethernet" mode.

Purpose: This is the foundation for Phase 21. The unified schema and store enable all downstream UI components (InputForm, BOMPanel, TopologyTab, etc.) to work with a single topology-aware store instead of separate Ethernet and Three-Tier stores. The mode type reduction from 4 to 3 values simplifies routing throughout the app.

Output: Unified SizingInputSchema with topology field, unified inputStore (v7), topology-aware resultStore, 3-button ModeSelector, updated i18n labels.
</objective>
