- Active tab underline/indicator
- Primary CTA button background (not secondary or ghost variants)

### Tabs

4 tabs in order: **Sizing** | Topology | Rack Elevation | Export

- "Sizing" tab is the default active tab on load
- "Topology", "Rack Elevation", "Export" tabs render a placeholder Card with disabled-state copy in Phase 2
- Tab bar scrolls horizontally on 768px viewports; no wrapping

### Responsive Breakpoints

| Viewport | Layout |
|----------|--------|
| 768px–1279px (tablet) | Input panel stacks above results panel (full width), separated by Separator component |
| 1280px+ (desktop) | Input panel (320px fixed) + results panel (flex-1) side-by-side |
| <768px | Out of scope (REQUIREMENTS.md "Out of Scope: Mobile") — no optimization required |

### Input Form Field Order

1. Total Server Count (number input, min: 1)
2. Servers per Rack (number input, min: 1, max: 48)
3. Connectivity Type (Select: 25G / 100G)
4. Cable Type (Select: DAC / AOC / Fiber)
5. Leaf Model (Select: S5248F-ON / S5224F-ON / S5212F-ON)

All 5 fields in a single Card with heading "Sizing Parameters". No submit button — every change triggers immediate recalculation.

## Interaction Contract

### Live Recalculation

- Debounce: 0ms (immediate) for Select and radio-style inputs
- Debounce: 150ms for number text inputs (avoids recalc on every keystroke mid-typing)
- Results panel shows previous result while recalculating (no flash/blank)
- No loading spinner needed — calculation is synchronous (<1ms for pure function)

### Inline Validation

- Error message renders below the field that fails validation, 12px red text, icon `AlertCircle` (lucide 14px)
- Error only shows after the field has been touched (blur) or after user clears a value to empty
- Valid state: no indicator — absence of error is sufficient (no green checkmarks)
- Required fields use `aria-required="true"` and `aria-describedby` pointing to error element

### Theme Toggle

- Icon: `Sun` (lucide) in dark mode, `Moon` (lucide) in light mode
- Button variant: ghost, size: icon (36px × 36px touch surface, inner icon 18px)
- Position: top-right of TopBar, left of language switcher
- System preference detected on first load via `window.matchMedia('(prefers-color-scheme: dark)')`
- Choice persisted to localStorage key `netstack-theme`

### Language Switcher

- DropdownMenu trigger showing active locale flag + code: "EN", "FR", "DE", "IT"
- Trigger variant: ghost, 44px minimum height
- Position: top-right of TopBar, rightmost element
- Choice persisted to localStorage key `netstack-locale`
- On language change: all labels, placeholders, error messages, tab names, placeholder copy update instantly — no page reload

### Placeholder Tabs

- Topology, Rack Elevation, Export tabs each render a Card with:
  - Icon: appropriate lucide icon (Network, Server, FileDown)
  - Heading: tab name (translated)
  - Body: placeholder copy (see Copywriting Contract)
  - Badge: "Coming in Phase N" (not translated — internal)

## Accessibility Contract

- All form inputs have explicit `<label>` elements linked via `htmlFor`/`id`
- Inline errors use `role="alert"` or `aria-live="polite"` for screen reader announcement
- Tab order: TopBar controls → Tab list → Form fields (top to bottom) → Results panel
- Color is never the sole indicator of state (error borders use icon + text + color)
- Focus ring: 2px solid accent color, 2px offset — never removed on keyboard focus
- ARIA landmark: `<header>` for TopBar, `<main>` for tab content area, `<aside>` for results panel
