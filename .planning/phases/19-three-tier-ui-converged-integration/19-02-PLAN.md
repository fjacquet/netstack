       **Y positions** (4-tier hierarchy + OOB):
       ```
       const Y = { core: 0, aggr: 160, access: 320, rack: 440, oob: 560 } as const
       ```

       **Layout strategy** (rack-based columns, same as Clos):
       - `RACK_COL_WIDTH = 320`, `NODE_GAP = 140`
       - `totalWidth = Math.max(bom.racks * RACK_COL_WIDTH, 600)`
       - `rackStartX`, `centerX`, `rackX` helpers identical to Clos builder

       **Core nodes** (centered at Y.core):
       - `coreSwitches` nodes with `role: 'core'`, model from `bom.input.coreModel`
       - `usedPorts`: `Math.ceil(bom.aggregationSwitches * bom.input.activeUplinksPerAggregation / bom.coreSwitches)`
       - `totalPorts`: `SWITCH_CATALOG[bom.input.coreModel].downlinkPorts`

       **Aggregation nodes** (centered at Y.aggr):
       - `aggregationSwitches` nodes with `role: 'aggregation'`, model from `bom.input.aggregationModel`
       - `usedPorts`: `Math.ceil(bom.accessSwitches * bom.input.activeUplinksPerAccess / bom.aggregationSwitches)`
       - `totalPorts`: `SWITCH_CATALOG[bom.input.aggregationModel].downlinkPorts`

       **Border leaf nodes** (same Y as access, positioned after last rack column):
       - Only if `bom.borderLeafSwitches > 0`, role `'border'`

       **Per-rack nodes** (same pattern as Clos but with `role: 'access'` instead of `'leaf'`):
       - Access pair A/B at Y.access with `role: 'access'`
       - OOB at Y.oob with `role: 'oob'`
       - Rack at Y.rack

       **Edges:**
       - access-aggr: each access switch connects to each aggregation switch (thin gray, strokeWidth: 1.5)
       - aggr-core: each aggregation switch connects to each core switch (thicker gray, strokeWidth: 2.5, to distinguish backbone)
       - border-core: each border leaf connects to each core switch (dashed, strokeDasharray: '8 4')
       - rack-access: each rack connects to its access pair (muted-foreground, strokeWidth: 1.5)
       - rack-oob: each rack connects to its OOB switch (muted-foreground, opacity: 0.5)
       - VLT: between access pairs (dashed blue, same as Clos VLT)

       Return `{ nodes, edges }` typed as `TopologyGraphResult`.
