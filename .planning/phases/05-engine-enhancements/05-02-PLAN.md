  <action>
    **Schema (src/domain/schemas/input.ts):**
    Add new field AFTER portsPerServerBackend:
    ```typescript
    /** Number of active uplinks per leaf switch to spine (1 to model max, default 4) */
    activeUplinksPerLeaf: z.number().int().min(1).max(8).default(4),
    ```
    Using `.default(4)` because S5248F-ON (most common leaf) has 4 uplink ports. The engine will clamp to the actual model's uplinkPorts at runtime.

    **Engine (src/domain/engine/sizing.ts):**
    Replace uplink calculation:
    - OLD: `const linksPerLeaf = Math.min(spineSwitches, LEAF.uplinkPorts);`
    - NEW:
    ```typescript
    // Runtime clamp: user cannot activate more uplinks than the model physically has
    const effectiveUplinks = Math.min(input.activeUplinksPerLeaf, LEAF.uplinkPorts);
    const linksPerLeaf = Math.min(spineSwitches, effectiveUplinks);
    ```

    Replace oversubscription calculation:
    - OLD: `const uplinkBandwidth = spineSwitches * (LEAF.uplinkSpeedGbE ?? 0);`
    - NEW: `const uplinkBandwidth = effectiveUplinks * (LEAF.uplinkSpeedGbE ?? 0);`

    This means oversubscription is now: `(maxServersPerRack * downlinkSpeed) / (effectiveUplinks * uplinkSpeed)` — the ratio reflects actual active uplinks, not total spine count.

    **Tests (src/domain/engine/sizing.test.ts):**
    Add new describe block `UPLN-02: Active Uplinks Per Leaf`:
    - `activeUplinksPerLeaf: 4 (default) matches previous behavior` — backward compat
    - `activeUplinksPerLeaf: 1 reduces leafSpineCables` — 2 leafs * min(2 spines, 1) = 2 cables
    - `activeUplinksPerLeaf: 1 increases oversubscription` — bandwidth = 1 * 100G = 100G only
    - `activeUplinksPerLeaf: 2 with 2 spines` — linksPerLeaf = min(2, 2) = 2 (same as min(2, 4) before)
    - `activeUplinksPerLeaf: 4 with S5212F-ON clamped to 3` — effectiveUplinks = min(4, 3) = 3
    - `QSFP28 transceivers scale with active uplinks` — fiber + reduced uplinks = fewer QSFP28

    Update ALL existing tests to include `activeUplinksPerLeaf: 4` (or verify makeInput helper includes it).
